Installation
============

``EmmyLua`` 是 `Jetbrains <https://www.jetbrains.com>`__ 产品家族的插件，使用 ``EmmyLua`` 之前请先安装 `Jetbrains <https://www.jetbrains.com>`__ IDE（建议使用 `IntelliJ IDEA Community <https://www.jetbrains.com/idea/download>`__），再用安装插件的方式将 ``EmmyLua`` 安装到IDE中使用

.. note::
	EmmyLua can be install on all product of `Jetbrains <https://www.jetbrains.com/>`_, Requires version ``2017.1`` or above.
