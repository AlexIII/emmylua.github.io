Installation
============

.. ``EmmyLua`` 是 `Jetbrains <https://www.jetbrains.com>`__ 产品家族的插件，使用 ``EmmyLua`` 之前请先安装 `Jetbrains <https://www.jetbrains.com>`__ IDE（建议使用 `IntelliJ IDEA Community <https://www.jetbrains.com/idea/download>`__），再用安装插件的方式将 ``EmmyLua`` 安装到IDE中使用

``EmmyLua`` is a plugin for `Jetbrains <https://www.jetbrains.com>`__ IDE, To use ``EmmyLua`` you should have installed `Jetbrains <https://www.jetbrains.com>`__ IDE (We recommend install `IntelliJ IDEA Community <https://www.jetbrains.com/idea/download>`__), and then install ``EmmyLua`` plugin to your IDE.

.. note::
	EmmyLua can be installed to all products of `Jetbrains <https://www.jetbrains.com/>`_, Requires version ``2017.1`` or above.
