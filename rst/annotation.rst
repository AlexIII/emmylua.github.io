.. 注解功能

Annotations
===========

.. EmmyLua 编辑器插件利用注解 ``annotation`` 功能来增加代码提示以及其它编辑器功能

EmmyLua use ``annotation`` to improve code completions and other features.

.. attention::

    EmmyLua注解功能只是单纯的辅助编辑器代码提示以及其它功能，和Lua代码的实际运行逻辑没有任何关系，因为它们就是普通的Lua注释
