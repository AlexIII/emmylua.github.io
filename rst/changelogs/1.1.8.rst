    .. seealso::
        :ref:`ann_dict`

    * 函数： ``fun(param:number):type``

    .. seealso::
        :ref:`ann_fun`

2. [NEW] 闭包函数参数类型推测

    .. code-block:: lua
        :linenos:
        :emphasize-lines: 4, 10

        ---@class Emmy
        ---@field name string

        ---@param callback fun(param:Emmy):void
        local function testCallback(callback)
            callback(nil)
        end

        testCallback(function(emmy) 
            -- emmy. 可以出现代码提示
        end)
