Run And Debug
=============

.. 1. 准备

1. Get ready
--------------

.. note::

    必须将源码的根目录设置为 ``Sources`` 目录。

    具体做法是打开菜单 ``File`` -> ``Project Structure`` 打开 ``Project Structure`` 设置面板，点击右侧的 ``Add Content Root`` 来添加你的源码根目录，然后点击 ``Mark as Sources`` 标记。
